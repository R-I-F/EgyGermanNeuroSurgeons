# عيادات مصرية التخصصية — Landing Page

Single-page promotional landing page for the visit of German Expert Neurosurgeon Prof. Henry Schröder. Built with **React (Vite)**, **Tailwind CSS**, and **Framer Motion**.

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
```

- Framer Motion
